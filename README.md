# Asistente Digitalizacion
Ejercicio de práctica en el que consite desarollar un pequeño asistente virtual para la asignatura de Digitalización aplicada a los sectores productivos en Segundo de DAM.

La práctica se centra en el desarollo del asistente y se realizan unos casos de uso de ejemplo.

Como añadido se desarolla un pequeño frontal en el que poder utilizar el asistente y se implementa un apartado con documentación. En él, se simula la parte de arquitectura en el desarollo de los casos de uso.
